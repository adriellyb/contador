## Informações Gerais

### Desafio

O desafio proposto é criar um contador (com dias, horas, minutos e segundos), a partir de uma data definida pelo usuário.
Todos os agradecimentos ao <a href="https://devchallenge.now.sh/">DevChallenge</a>.
